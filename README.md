# Discord Rpc
> Set your Discord RPC from android - no dekstop app required.

> works on android ( termux, smartIDE, etc )
> no discord dekstop app needed.


## Activity Types
Change `"type"` in the script to switch the display:

| Number | Shows as |
|--------|----------|
| `0` | Playing |
| `1` | Streaming |
| `2` | Listening to |
| `3` | Watching |
| `4` | Custom (doesn't work) |
| `5` | Competing in |



## install the dependency

```bash
pip install websocket-client
```
